# VAD-Based Speech-to-Text Web App

A web application for transcribing audio using Whisper Large-v3-Turbo with 8kHz mono WAV format.

## Installation

1. Install dependencies:
```bash
./setup.sh
```

## Usage

1. Start the Flask application:
```bash
python app.py
```

2. Open your browser and navigate to:
```
http://localhost:5000
```
